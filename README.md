# restaurant-website-template

A simple and clean restaurant website template built with **HTML and CSS**.  
This project is suitable for small food businesses, home kitchens, and beginners learning web development.

## Features
- Responsive layout
- Menu section with images
- Order section
- WhatsApp order button
- Clean and simple design

## How to Use
1. Download or clone this repository
2. Open `index.html`
3. Edit the text to match your business
4. Replace images inside the `images` folder
5. Update the WhatsApp number in the footer

## Technologies Used
- HTML5
- CSS3
